# 会社マスターExcel サンプル

このフォルダには、システムで使用する「会社マスター.xlsx」のサンプルファイルを配置してください。

## ファイル構成

```
会社マスター.xlsx
├── シート1: 会社マスタ
├── シート2: メール  
└── シート3: 保存先（オプション）
```

---

## シート1: 会社マスタ

会社の基本情報を登録します。

| A列: 会社名 | B列: メールアドレス | C列: CC |
|------------|-------------------|---------|
| サンプル株式会社 | invoice@sample.co.jp | |
| テスト工業株式会社 | accounting@test.jp | manager@test.jp |

### 注意事項
- A列の会社名は、請求書PDFに記載されている会社名と完全一致する必要があります
- B列のメールアドレスは必須です
- C列のCCは任意です（複数の場合はセミコロン区切り）

---

## シート2: メール

メールテンプレートを設定します。

```
A1: メールタイトル
A2: 【会社名】YYYY年MM月分 ご請求書送付の件
A3: (空行)
A4: メール本文
A5: A:A
A6: ご担当者様
A7: 
A8: いつもお世話になっております。
A9: 株式会社○○の△△でございます。
A10: 
A11: YYYY年MM月分のご請求書を送付いたします。
A12: ご査収のほど、よろしくお願いいたします。
A13: 
A14: ご不明な点がございましたら、お気軽にお問い合わせください。
A15: 
A16: 以上、よろしくお願いいたします。
```

### プレースホルダー
- `A:A` → 会社名に自動置換されます
- `YYYY年MM月` → 締め日から自動抽出されます（例: 2025年12月）
  - 月の0埋めはありません（1月、12月）

---

## シート3: 保存先（オプション）

PDFの保存先パスを設定します。

| A列 | B列 |
|-----|-----|
| フォルダ | \\server\shared\invoices |

### 形式
- ローカルパス: `C:\請求書`
- ネットワークパス: `\\server\shared\invoices`

### 動作
- B1セルにパスが設定されている場合、自動的にそのパスを使用
- 未設定の場合、手動でフォルダ選択ダイアログが表示されます
- 指定パス配下に、自動で「YYYY年MM月」フォルダが作成されます

---

## サンプルファイルのダウンロード

GitHub Releasesから`会社マスター_サンプル.xlsx`をダウンロードできます。

---

## トラブルシューティング

### Q. 会社名が認識されない
A. 請求書PDFの会社名と、Excel A列の会社名が完全一致しているか確認してください

### Q. メールテンプレートが正しく読み込まれない
A. シート名が「メール」になっているか確認してください

### Q. 保存先が反映されない
A. シート名が「保存先」になっており、B1セルにパスが入力されているか確認してください
